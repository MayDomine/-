## 键位映射
### Windows
|新键位|原键位|
|:----------:|:--------------:|
|alt + hjkl|left down up right|
|caps + h l|ctrl + left/right|
|ctrl + [ |home|
|ctrl + ] |end|



### Mac
|新键位|原键位|
|:----------:|:---------------:|
|caps + hjkl|left down up right|
|caps + shift|caps|
|caps + [ |home|
|caps + ] |end|
|caps (alone) |Escape|
